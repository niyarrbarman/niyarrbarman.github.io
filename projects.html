Projects

  • DiagnoAI

    DiagnoAI is a tool to detect a disease from a text description of the patient's symptoms. It generates a probability score for the top 3 of the 24 common diseases it has been fine-tuned to detect. The BERT transformer is used as the base architecture, and fine-tuned for this specific task to improve its performance.

  • CalmSpace

    CalmSpace is a sentiment analysis platform that uses machine learning to predict emotions from audio recordings. The website presents results through graph plots and the model is integrated using Flask API. It can be used by individuals, therapists, and researchers to better understand emotions in speech.

       Repository
  • Floodment

    FloodMent is a deep learning project that uses aerial images and segments flooded areas. It utilizes a fully Convolutional U-Net architecture and achieved a training IoU score of 0.7991 and validation IoU score of 0.7103. It can be used in flood monitoring for quick identification of affected areas.

       Repository
  • PneumoPred

    PneumoPred is a deep learning model that predicts pneumonia from chest x-ray images. It uses transfer learning and fine-tunes a pre-trained ResNet-18 model using Pytorch, achieving a validation accuracy of 0.625 and testing accuracy of 0.8958. It can be used in medical settings to aid in the diagnosis of pneumonia.
